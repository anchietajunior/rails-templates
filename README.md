# Rails-Templates

Create Ruby on Rails applications faster than it already is.

## TLDR Template

This template includes:

- [x] Rspec-Rails
- [x] Shoulda Matchers
- [x] Database Cleaner
- [x] Faker
- [x] FactoryBot
- [x] Devise

### How to use it?

```shell
rails new -T -d postgresql -m ../path/to/template/tldrtemplate.rb
```
